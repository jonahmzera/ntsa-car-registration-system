# NTSA Car Registration System (C++)

This is a console-based system built using C++ and OOP principles.

## Features
- Register multiple vehicles
- Store client and vehicle details
- Calculate NTSA charges (KES 1000 per passenger)
- Display structured records

## How to Run

1. Install a C++ compiler (e.g., g++)
2. Compile:
   g++ main.cpp -o ntsa
3. Run:
   ./ntsa
